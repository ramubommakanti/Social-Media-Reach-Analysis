# Social Media Reach Analysis

## Objective
Analyze social media reach data using Microsoft Excel.

## Tools Used
- Microsoft Excel

## Dataset
The dataset contains:
- Date
- Social Media Platform
- Reach
- Likes
- Shares
- Impressions

## Steps Performed
1. Created and organized the dataset.
2. Created a PivotTable to summarize total reach by platform.
3. Created a PivotChart to visualize the results.
4. Analyzed the performance of each social media platform.

## Output
The project includes:
- Original dataset
- PivotTable
- PivotChart titled "Social Media Reach Analysis"
